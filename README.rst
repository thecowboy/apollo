======
Apollo
======

Depends
-------

 * Python (2.6+)
   http://www.python.org

 * Tornado
   https://github.com/facebook/tornado

 * MongoDB
   http://www.mongodb.org

 * Ming
   http://merciless.sourceforge.net

Installation
------------

Edit the ``apollod.conf`` file as appropriate. See ``apollod.py --help`` for
information on the options.
